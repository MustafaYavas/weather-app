# weather-app

## About
***A simple app to get current weather***

Technologies
* Javascript - Node.js

APIs that used
* [weatherstack.com](https://weatherstack.com/)
* [mapbox.com](https://www.mapbox.com/)

## Usage
Install Packages
>npm install

Open [http://localhost:3000](http://localhost:3000/) to run app
>npm run start
