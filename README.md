# weather-app

## About
A simple weather forecast application. [Try It](https://mustafa-weather-forecast-app.herokuapp.com/)

Technologies
* Javascript - Node.js

APIs that used
* [weatherstack](https://weatherstack.com/)
* [mapbox](https://www.mapbox.com/)

## Usage
Install Packages
>npm install

Open [http://localhost:3000](http://localhost:3000/) to run app
>npm run start
